# AdoptME

## Опис проєкту
AdoptME - це спеціалізована онлайн-платформа для адопції безпритульних тварин в Україні. Сервіс об'єднує притулки, волонтерів та людей, які шукають домашніх улюбленців, з метою спрощення процесу адопції та зменшення кількості безпритульних тварин.

## Технології

**CSS Flexbox** - система макетування для створення гнучких та адаптивних розкладок. Використано для розташування фільтрів та списку тварин, вирівнювання логотипу та навігації, горизонтального меню, організації футера.

**CSS Grid** - система сіткової розкладки для складних макетів. Використано для відображення карток тварин у вигляді адаптивної сітки.

**Media Queries (@media)** - умовні стилі для різних розмірів екранів. Використано для мобільної версії (зменшення шрифтів, адаптація навігації) та планшетної версії (відображення карток у два стовпці).

**HTML5 Form Validation** - вбудована валідація форм через атрибути. Використано для обов'язкових полів, перевірки формату email, мінімальної довжини паролів, візуальної стилізації помилок та валідних полів.

**CSS Custom Properties** - змінні для збереження повторюваних значень. Використано для кольорів бренду, базових кольорів та границь.

**CSS @import** - директива для об'єднання CSS файлів. Використано для імпорту базових стилів, компонентів та макетів в один головний файл.

**CSS Transitions** - плавні переходи між станами. Використано для ефектів hover на кнопках, посиланнях та картках.

**CSS Box Shadow** - тіні для елементів. Використано для карток тварин, хедера та форм авторизації.

**CSS Transform** - трансформації елементів. Використано для ефекту підняття елементів при наведенні курсора.
